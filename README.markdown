# Plus-4

This is a pretty simple terminal emulator.

## Keyboard Shortcuts

 - **ctrl** + **t** - new terminal
 - **ctrl** + **w** - close terminal
 - **ctrl** + **→** - next terminal
 - **ctrl** + **←** - previous terminal

 - **ctrl** + **shift** + **c** - Copy
 - **ctrl** + **shift** + **v** - Paste

 - **ctrl** + **+** - increase the font size
 - **ctrl** + **-** - decrease the font size
 - **ctrl** + **+** - reset the font size
